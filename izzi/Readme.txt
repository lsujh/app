Завдання:
Основні вимоги:
Створити проект за допомогою фреймворку Django.
Додати можливість завантажити інформацію про користувачів в базу даних з файлу формату csv самостійно.2
Парсер CSV потрібно написати самому
Створити API за допомогою DRF (Django REST framework) для отримання даних в форматі json:
всіх користувачів;
користувачів, зареєстрованих певного числа (іншими словами фільтрація по даті реєстрації);

Додати до проекту файл Readme з покроковою інструкцією запуску проекту та процесу імпорту користувачів, отримання їх через АПІ.я

Опис проекту:
Основа проекту: база даних користувачів та їх замовлень.
Модель Користувач:
Поля: Ім’я, Прізвище, Дата народження, Дата реєстрації, Замовлення(зв’язок з моделлю замовлення)
Зв’язок: Користувач може мати лише одне замовлення, або не мати зовсім.
Модель Замовлення:
Поля: Дата створення замовлення, Товар
Встановлення зв’язку між моделями, а також обрання типу даних для полів залишається на Ваш розсуд.
Можливість завантаження користувачів з файлу:
В адмін панелі має бути передбачена можливість завантажити інформацію про користувачів з csv - файлу (файл TestData.csv)

Термін виконання: 2-3 дні.



Запуск проекту:
1) git clone https://github.com/lsujh/izzi.git
2) cd izzi
3) python -m venv venv
4) source venv/bin/activate 
5) pip install -r requirements.txt
6) python manage.py makemigrations
7) python manage.py migrate
8) python manage.py createsuperuser
9) python manage.py runserver


Початок роботи:
Вхід:
1) Перейти за адресою ʼадреса сайту/admin/ʼ
2) Ввести І'мя Користувача та пароль.

Створення одного користувача:
1) Натиснути ʼ+Додатиʼ напроти напису ʼКористувачіʼ.
2) Заповнити поля.
3) Зберегти.

Завантаження списку користувачів з файлу csv:
1) Натиснути ʼКористувачіʼ.
2) Натиснути ʼЗавантажити CSVʼ
3) Натиснути ʼВиберіть файлʼ
4) У вікні, яке відкриється, вибрати потрібний файл.
5) Натиснути кнопку ʼЗавантажити CSVʼ

Отримання списку користувачів через АПІ:
Всіх користувачів:
1) Перейти за адресою ʼадреса сайту/api/users/ʼ

Користувачів, зареєстрованих певного числа:
1) Перейти за адресою ʼадреса сайту/api/users/YYYY-MM-DDʼ
замість ʼYYYY-MM-DDʼ вставити потрібні цифри.
